# 📊 The Future of Employment: Global Market Analytics

An interactive Excel data visualization project analyzing the shifting landscape of the global job market. This dashboard provides key insights into emerging high-growth roles, work hours, salary distributions across industries, and the impact of academic backgrounds on hiring trends.

---

## 🚀 Key Insights & Features

*   **📈 High-Growth Sectors:** Tracks the fastest-growing tech and healthcare roles, highlighting high-demand professions like Blockchain Developers, Software Developers, and Data Scientists.
*   **⏰ Work-Life Balance Metrics:** Analyzes weekly work hours across top-tier technical roles to identify demanding industry benchmarks.
*   **💰 Industry Compensation:** Compares average expected salaries across major sectors including Finance, Technology, Media, and Transportation.
*   **🎓 Academic Demographics:** Features an intuitive breakdown of job availability mapped against education levels (Bachelor, Master, PhD, and High School).
*   **🌍 Global Filtering:** Built-in multi-country slicers supporting regional analysis across Germany, India, Japan, UK, USA, and more.

---

## 🛠️ Tech Stack

*   **Data Visualization & Modeling:** Microsoft Excel (Advanced Charts, Pivot Tables, Slicers)
*   **Data Source:** Kaggle
*   **Concepts Used:** Data Cleaning, Pivot Architecture, Interactive Slicers, Dashboard UI/UX Design
